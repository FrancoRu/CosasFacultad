Para esta práctica se utilizara la herramienta de simulación de redes de CISCO Packet Tracer

Al finalizar la práctica deberemos tener configurado una topología como la siguiente

 

 

 

 Tarea 1: Configurar la interfaces según la topología anterior. 

Tarea 2: Configurar la interfaces Seriales de los Router con la siguiente topología

La conexión entre el Router_Red1 y el Router_Red2 utilizan conexión PPP con autenticación Chap.

1.     Colocamos el nombre Router_Red1 al Router de la Red1y Router_Red2 al de la red 2

 

2.     Configuramos la clave para la autenticación chap con el comando

#username Router_Destino password Contraseña

   

3.     Levantar las interfaces seriales de los dispositivos.

 

4.     Estas interfaces pertenecen a la red 10.0.0.0/30 donde el Router_Red1 utiliza la 1ra IP (10.0.0.1/255.255.255.252) y la última para el Router_Red2 (10.0.0.2/255.255.255.252)

 

5.     Configuramos la encapsulación PPP en ambos Routers.

#encapsulation ppp (Esto se realiza dentro de la interfaz que vamos a configurar)

 

6.     Configuramos autenticación CHAP en los Routers.

#ppp authentication chap (Esto se realiza dentro de la interfaz que vamos a configurar)

 

4.      La conexión entre el Router_Red1 y el Router_Red3 utilizan conexión HDLC2

1.     Colocamos el nombre Router_Red3 al Router de la Red3

 

2.     Configuramos la encapsulación HDLC en el puerto serie correspondiente en ambos Routers.

#encapsulation hdlc  (Esto se realiza dentro d el ainterfaz que vamos a configurar)

 

3.     Estas interfaces pertenecen a la red 10.0.0.4/30 donde el Router_Red1 utiliza la 1ra IP (10.0.0.5/255.255.255.252) y la última para el Router_Red3 (10.0.0.6/255.255.255.252).

 

 

Tarea 3: Configurar el Gateway apropiado en cada uno de los equipos (PCs) para que se puedan comunicar con todas las redes.

 

Tarea 4: Configurar las Rutas Estáticas en los routers para que se comuniquen las distintas redes.

Ejemplo

#ip route 10.0.0.1 255.255.255.0 10.2.0.1

 

Tarea 5: Pruebas de conexión entre los distintos equipos de las distintas redes.

 

Tarea 6: Nos colocamos en Modo de Simulación

Ejecutamos un Ping de la PC1_Red3 a la PC1_red1

Presionamos Auto Capture / Play

Verificamos que estén correctamente configurados los protocolos en cada uno de los routers.
