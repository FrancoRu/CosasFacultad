Tarea 2  Conexión y configuración Oficina 1 y DMZ

Paso 1  Configurar el direccionamiento y/o problemas de capa física

a)      Controlar y solucionar problemas de capa física de los dispositivos pertenecientes a Oficina 1 y DMZ. El Router0 se comunica a la Nube mediante la interfaz Fa0/0 a Eth6 -nube-

b)      Configurar el direccionamiento IP en base a la tabla de direccionamiento

Paso 2. Configure nombre y seguridad a los dispositivos intermedios

a)      Configure el nombre los dispositivos finales e intermedio utilizando la Tabla de Direcciones 

b)      En los dispositivos intermedios configure:

·                    Configure la seguridad de modo EXEC privilegiado en modo secreto (clave = clase).

·                    Además configure la seguridad de modo acceso a consola y las línea vty (clave = red).

c)      Encriptar las contraseñas que se encuentren en texto plano

Paso 3. Configure direccionamiento estático en ROficina1

Utilizando el comando ip route en el modo de configuración global configure una ruta estática con los siguientes parámetros:

·         Dirección Red: 0.0.0.0

·         Mascara: 0.0.0.0

·         Interfaz Salida: serial 0/0/0

 

Tarea 3  Conexión y configuración Oficina 2

Paso 1  Configurar el direccionamiento y/o problemas de capa física

c)      Controlar y solucionar problemas de capa física de los dispositivos pertenecientes a la Oficina 2.

d)      Configurar el direccionamiento IP en base a la tabla de direccionamiento

Paso 2. Configure nombre y seguridad de Router1

d)      Configure el nombre los dispositivos finales e intermedio utilizando la Tabla de Direcciones 

e)      En los dispositivos intermedios configure:

·                    Configure la seguridad de modo EXEC privilegiado en modo secreto (clave = clase).

·                    Además configure la seguridad de modo acceso a consola y las línea vty (clave = red).

f)       Encriptar las contraseñas que se encuentren en texto plano
